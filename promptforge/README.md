# PromptForge — AI Prompt Engineering Studio

A purpose-built studio to design, analyze and optimize AI prompts.

## Features

- ✍️ **Smart Editor** — Distraction-free writing with auto-save
- 📊 **Real-time Analysis** — Quality scores for clarity, specificity, structure & context
- 🎯 **Actionable Suggestions** — Severity-ranked improvement tips
- 📚 **Template Library** — 8+ curated prompts across Code, Content, Analysis, Creative, Learning & Marketing
- ⚡ **One-click Optimize** — Auto-enhance prompts with best practices
- ⭐ **Favorites & Search** — Organize your prompt collection
- 💾 **Local Persistence** — Zustand with localStorage

## Tech Stack

React 18 · Vite · Zustand · Tailwind CSS · Lucide Icons

## Dev

```bash
npm install
npm run dev
```
